# Project Wiki

## Overview
A concise introduction to the `javaconcepts` project: learning examples for Java features (lambdas, streams, sorting, value objects, tests).

## Getting Started
1. Prerequisites: Java 11+ and Maven.
2. Build: `mvn clean package`
3. Run tests: `mvn test`

## Project Structure
- `src/main/java` — application examples and demos
- `src/test/java` — unit tests
- `pom.xml` — build configuration

## How to Run Examples
Run compiled classes or use an IDE (IntelliJ/Eclipse) to run `main` classes. Example modules:
- `LambdaFunctions.java` — lambda and stream examples
- `SortingConcept.java` — sorting examples
